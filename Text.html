Data Base Management System

Une base de données est une collection organisée de données , généralement stockées et consultées électroniquement à partir d'un système informatique. Lorsque les bases de données sont plus complexes, elles sont souvent développées à l'aide de techniques formelles de conception et de modélisation .

Le système de gestion de base de données (SGBD) est le logiciel qui interagit avec les utilisateurs finaux , les applications et la base de données elle-même pour capturer et analyser les données. Le logiciel SGBD comprend en outre les fonctionnalités de base fournies pour administrer la base de données. La somme totale de la base de données, du SGBD et des applications associées peut être appelée «système de base de données». Souvent, le terme «base de données» est également utilisé pour désigner de manière approximative l'un des SGBD, le système de base de données ou une application associée à la base de données.

Les informaticiens peuvent classer les systèmes de gestion de bases de données en fonction des modèles de bases de données qu'ils prennent en charge.
